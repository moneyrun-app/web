## 2. 서비스 비전 & 핵심 메시지

> **"부자 되세요"가 아니라 "거지 되지 마세요"**

복잡한 투자·시뮬레이션 대신, 매일 AI가 한마디씩 잔소리해주는 구조. 유저가 매일 앱을 열면 "오늘 너 얼마 쓸 수 있어", "이번 주 이거 좀 많이 썼다?", "이 글 읽어봐" 같은 메시지를 받는다.

핵심은 **습관**이다. 한 번 진단하고 끝나는 게 아니라, 매일 돌아와서 페이스메이커 메시지를 확인하는 루틴을 만드는 것.

---

## 3. 타겟 유저

20대 중후반 ~ 30대 초반 사회초년생. 돈을 벌기 시작했지만 어떻게 관리해야 하는지 모르는 사람. "투자 앱은 무섭고, 가계부는 귀찮은" 사람.

## 4. MVP 범위 — 뭘 만들고 뭘 안 만드는가

### MVP에 포함

| 기능 | 설명 |
|---|---|
| **온보딩** | 나이, 월 실수령액, 좋은 소비, 고정 소비 입력 → 하루 사용 가능 금액 자동 계산 |
| **페이스메이커** | AI가 하루 1번 맞춤 메시지 생성 (홈 화면) |
| **마이북** | 카테고리별 콘텐츠 모음 (상세 리포트, 주간 리포트, 스크랩, 운영팀 콘텐츠) |
| **마이페이지** | 내 정보 수정 (수정 시 잉여자금 재계산), 계정 설정 |

### MVP에서 제외 (추후 로드맵)

| 기능 | 제외 이유 |
|---|---|
| 커뮤니티 | 유저 수 확보 후 의미 있음 |
| 자산 트래킹 | 핵심 가치(페이스메이커)에 집중 |
| 65세 시뮬레이션 독립 페이지 | 페이스메이커가 참조 데이터로 활용 (별도 페이지 불필요) |
| AI PDF 리포트 이메일 발송 | 마이북 상세 리포트로 대체 |
| 주간 체크인 UI | 주간 리포트는 자동 생성, 별도 체크인 화면 불필요 |
| 소득 구간 분류 | 커뮤니티와 함께 추후 |

## 5. 로그인 정책 & 온보딩

### 로그인

카카오 소셜 로그인 단일. 비로그인 상태는 없음 — 앱 진입 시 바로 로그인.

### 온보딩 (로그인 직후, 최초 1회)

간단한 정보 입력 → 잉여자금(하루 사용 가능 금액) 즉시 계산.

**입력 항목:**

| 순서 | 항목 | 입력 방식 | 필드명 |
|---|---|---|---|
| 1 | 현재 나이 | 숫자 입력 (만 나이) | `age` |
| 2 | 월 실수령액 | 숫자 입력 (원 단위) | `monthlyIncome` |
| 3 | 좋은 소비 | 항목별 금액 입력 | `goodSpendings[]` |
| 4 | 고정 소비 | 항목별 금액 입력 | `fixedExpenses` |

**좋은 소비 항목:**

| 코드 | 라벨 | 설명 |
|---|---|---|
| `savings` | 적금 | 은행 적금 |
| `investment` | 투자 | 주식·ETF 등 |
| `pension_savings` | 연금저축 | 연금저축펀드 등 |
| `irp` | IRP | 개인형퇴직연금 |
| `insurance` | 보험 | 보험료 |

**고정 소비 항목:**

| 코드 | 라벨 | 설명 |
|---|---|---|
| `rent` | 월세 | 0원 가능 (부모님 집 등) |
| `utilities` | 관리비+공과금 | |
| `phone` | 통신비 | |

**온보딩 결과:**

```
잉여자금 = 월 실수령액 - 좋은 소비 합계 - 고정 소비 합계
하루 사용 가능 = 잉여자금 / 30

→ "너 하루에 OO만 원 쓸 수 있어"
```

온보딩 완료 즉시 → 페이스메이커(홈)로 이동.

## 6. 전체 서비스 구조

### 3페이지 구조

```
[카카오 로그인]
  → [온보딩] (최초 1회)
  → [페이스메이커(홈)] ← 메인 화면, 매일 돌아오는 곳

하단 3탭:
  페이스메이커  |  마이북  |  마이
```

| 탭 | 라우트 | 핵심 역할 |
|---|---|---|
| **페이스메이커** | `/home` | AI 일일 메시지 + 하루 사용 가능 금액 + 오늘의 한마디 |
| **마이북** | `/book` | 상세 리포트 / 주간 리포트 / 스크랩 / 학습 콘텐츠 (카테고리별, 날짜별) |
| **마이** | `/my` | 내 정보 수정 + 계정 설정 |

## 7. 잉여자금 자동 계산 시스템

### 핵심 공식

```
좋은 소비 합계 = sum(goodSpendings[].amount)
고정 소비 합계 = rent + utilities + phone

잉여자금(월) = monthlyIncome - 좋은 소비 합계 - 고정 소비 합계
잉여자금(주) = 잉여자금(월) / 4.3
잉여자금(일) = 잉여자금(월) / 30
```

### 재계산 트리거

마이페이지에서 어떤 항목이든 수정하면 → 즉시 전체 재계산.

---

## 8. 페이스메이커 (홈)

AI가 먼저 말을 거는 구조. 유저가 물어봐야 답하는 챗봇이 아니라, **매일 1번 메시지를 생성**해서 보여준다.

### 표시 정보

1. **오늘의 AI 메시지** (핵심, 화면 상단 고정)
2. 하루 사용 가능 금액 ("오늘 OO원 쓸 수 있어")
3. 신호등 등급 뱃지

### 메시지 톤 가이드

- **찐친이 잔소리** — 걱정하면서도 직접적으로 말하는 톤
- 모든 메시지 끝은 **행동 유도**
- 등급별 톤 차이:
  - RED: 직접적 + 유머
  - YELLOW: 은근히 찌르기
  - GREEN: 격려 + 다음 단계

### 메시지 하단 — 추천 행동

AI 메시지 아래에 1~2개 추천 행동 표시 → 클릭 시 마이북 해당 콘텐츠로 이동

## 9. 마이북

내 금융 관련 콘텐츠가 모이는 공간. 카테고리별로 정리되고, 날짜별로 쌓인다.

### 카테고리 구조

| 카테고리 | 설명 |
|---|---|
| **상세 리포트** | AI가 생성한 내 재무 분석 리포트 (프로필 변경 시 트리거, 월 1회 제한) |
| **주간 리포트** | 유저 입력 기반 → AI가 한 쪽짜리 가이드 생성 |
| **스크랩** | 유저가 직접 스크랩한 콘텐츠 모음 |
| **금융 학습** | 운영팀이 등록한 등급별 금융 교육 콘텐츠 |

---

## 10. 마이페이지

1. **내 재무 정보 수정** — 수정 즉시 잉여자금 재계산
2. **계정 설정** — 알림, 마케팅 동의
3. **카카오 연동 정보**
4. **회원 탈퇴**

---

## 11. 신호등 등급 시스템

```
좋은 소비 비율 = 좋은 소비 합계 / 월 실수령액 x 100

RED    : 좋은 소비 = 0원 OR 비율 < 10%
YELLOW : 10% <= 비율 < 20%
GREEN  : 비율 >= 20%
```
